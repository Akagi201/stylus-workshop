# Counter

## Init

```sh
cargo stylus new counter
```

## Build

```sh
cargo build
```

## Generate ABI

```sh
cargo stylus export-abi
```

## Check wasm

```sh
cargo stylus check
```

## Deploy

estimate gas

```sh
cargo stylus deploy --private-key xxx --estimate-gas-only
```

true deploy

```sh
cargo stylus deploy --private-key xxx
```
